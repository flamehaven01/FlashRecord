Features
--------

* **Screenshot Capture**: Native cross-platform screenshot capture
* **GIF Recording**: Screen recording to animated GIF with CWAM compression
* **PNG Compression**: Intelligent resolution scaling (Phase 1+2)
* **CLI Interface**: Simple command-line interface
* **Docker Support**: Containerized deployment
* **CI/CD**: Automated testing and deployment
